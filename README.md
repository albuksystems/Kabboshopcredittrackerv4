Deploy to GitHub Pages
Go to github.com → New repository → name: KabboCredit → Public
Click Add file → Upload files
Upload these 3 files:
kabbo_credit_tracker.html → rename to index.html
manifest.json
sw.js
Commit changes
Go to Settings → Pages → Branch: main → Save
Wait 2 minutes → live at albuksystems.github.io/KabboCredit/
Install on Phone
Android (Chrome)
Open Chrome → go to your GitHub Pages URL
Tap 3-dot menu → Add to Home Screen
Tap Add → app appears on home screen
iPhone (Safari)
Open Safari → go to your GitHub Pages URL
Tap Share button → Add to Home Screen
Tap Add → app appears on home screen
Backup Your Data
Go to Settings → Data Backup
Enter your Gmail address
Tap Backup to Gmail Now
Gmail opens → hit Send
A .json file also downloads to your phone
Reminder appears every 3 days automatically.
